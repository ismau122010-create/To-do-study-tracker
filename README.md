# To-Do & Study Tracker (CLI)

A simple command-line Python project that lets you manage tasks and track study sessions.  
Data is saved locally using a JSON file so it persists between runs.

## Features
- Add tasks
- View all tasks
- Mark tasks as done / not yet
- Delete a task or delete all data
- Add study sessions (tracks total duration per subject)
- View study report
- Save & load data using JSON

## How to Run
1. Make sure you have Python installed.
2. Download/clone this repository.
3. Run the program:

```bash
python test1.py
